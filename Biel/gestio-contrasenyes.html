Gestió de Contrasenyes en Python

Descripció: Aquest projecte permet emmagatzemar i gestionar contrasenyes de manera segura utilitzant el llenguatge Python i una base de dades senzilla. El sistema implementa la xifratge de contrasenyes per garantir la seguretat de la informació.

Tecnologies utilitzades:

  • Python
  • SQLite
  • Cryptography

Detalls del projecte:

Aquesta aplicació utilitza SQLite com a base de dades per emmagatzemar les contrasenyes de manera encriptada. S'ha implementat una interfície d'usuari senzilla amb una llista de contrasenyes que l'usuari pot afegir, eliminar o actualitzar.
